Research Snapshot

My recent work spans risky-text detection, empathetic dialogue systems, machine learning for quantitative trading, and Android security. Across these topics, I care about robust evaluation, reproducibility, and meaningful system design.

Risky Text Detection Benchmark

Built a benchmark spanning spam, phishing, and fraud-like messages, with multi-baseline comparison, cross-domain evaluation, and robustness-oriented analysis.

Empathetic Dialogues Program

Developed an empathetic dialogue system on DeepSeek with a modular Chain-of-Empathy component to improve emotional awareness and response controllability.

Quantitative Trading Research

Studied reinforcement learning for cross-sectional trading through reproducible pipelines, configurable environments, backtesting, and sensitivity analysis.

Android Security & API Analysis

Supported Java-based API requirement analysis, attack–defense experiments, and system-level testing for Android security research at HKU.
