# GitHub Pages Homepage Draft

This package contains a first draft of an academic personal homepage.

## Files
- `index.html` — main homepage file
- `profile.jpg` — profile photo used on the homepage
- `Zhang_Yuchen_CV.pdf` — CV download file

## How to deploy
1. Create a GitHub repository named `YOUR_USERNAME.github.io`
2. Upload all files in this folder to the repository root
3. Commit and push
4. In GitHub: **Settings → Pages**
5. Set source to **Deploy from a branch**
6. Choose branch `main` and folder `/root`
7. Open `https://YOUR_USERNAME.github.io`
